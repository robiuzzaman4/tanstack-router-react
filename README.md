## **TanStack Router React**

A starter pack of TanStack Router with React.js & Vite

### **TeachStacks**

- **React.js**
- **TanStack Router**
- **Tailwind CSS**
- **Shadcn UI**

```bash
# install dependencies
pnpm install
```

```bash
# runing development server
pnpm run dev
```

```bash
# building application
pnpm run build
```
